# Model
The EcoGP model can be found here [EcoGP/model.py](model.py) with the different likelihoods located in [EcoGP/likelihoods.py](likelihoods.py).

## Training
To train the model, the file [EcoGP/train.py](train.py) can be ran by "python train.py --config <config_file>", which loads a config file from the [configs/](../configs/) folder.
